```
C-a " = list
C-a n = next
C-a p = previous
C-a c = create
C-a <number> = specific window
C-a [ = enter copy mode, space to start, space to end, C-a ] to paste
C-a S = split horizontal
C-a | = split vertical
C-a <tab> = move between splits
C-a X = remove current split
C-a Q = remove other split
```
